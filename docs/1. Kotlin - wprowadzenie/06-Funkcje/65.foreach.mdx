# `forEach()`
- w Kotlinie działa bardzo podobnie jak w TypeScripcie czy C#
- to metoda kolekcji, która pozwala wykonać daną akcję dla każdego elementu listy, zbioru czy mapy.

## Podstawowe zastosowanie
```kotlin
  val numbers = listOf(1, 2, 3, 4, 5)

    numbers.forEach { number ->
        println("Liczba: $number")
    }
/*
    można skrócić do:
    numbers.forEach { println("Liczba: $it") }
*/

// zapis równoważny
 for(number in numbers) {
      println("Liczba: $number")
  }
```


## Przykład z indeksem
```kotlin
val fruits = listOf("apple", "banana", "cherry")

	fruits.forEachIndexed { index, fruit ->
   		println("${index + 1}. $fruit")
	}
```

## Przykład ze słownikiem/mapą
```kotlin
val people = mapOf("Alice" to 25, "Bob" to 30)

people.forEach { (name, age) ->
    println("$name ma $age lat")
}
```

## Przykład z obiektem
````kotlin
data class Person(val name: String, val age: Int, val job: String)

fun main() {
    val people = listOf(
        Person("Alice", 25, "Engineer"),
        Person("Bob", 30, "Teacher"),
        Person("Charlie", 22, "Designer")
    )

    // Dla każdego elementu z listy wykonaj kod w lambda
    people.forEach { person ->
        println("${person.name} ma ${person.age} lat i pracuje jako ${person.job}.")
    }
}

// w wersji skróconej:
    people.forEach {
        println("${it.name} ma ${it.age} lat i pracuje jako ${it.job}.")
    }
}

// z warunkiem
people.forEach {
    if (it.age >= 30) {
        println("${it.name} jest dorosły i ma ${it.age} lat.")
    }
   }


//z indeksem
 	people.forEachIndexed { index, person ->
    	println("${index + 1}. ${person.name} (${person.job})")
	}
```
