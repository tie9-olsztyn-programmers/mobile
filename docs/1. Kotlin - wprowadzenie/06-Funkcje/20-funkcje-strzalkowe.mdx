# Definicja

Funkcje strzałkowe, inaczej zwane wyrażeniami lambda (lambda expressions), to:
- anonimowe funkcje, które można
    - tworzyć i
    - przekazywać
    - jako parametry w Kotlinie.
- Używa się ich, gdy potrzebujesz krótkich, jednorazowych funkcji, które nie wymagają pełnej definicji za pomocą słowa kluczowego fun.

# Podstawowa składnia

`{ parametry -> wyrażenie }`
- *parametry*: Lista parametrów funkcji lambda (może być pusta).
- *wyrażenie*: Kod, który zostanie wykonany w ciele funkcji i zazwyczaj zwrócony jako wynik.

# Przykłady
## Prosta funkcja strzałkowa
```kotlin
val dodaj = { a: Int, b: Int -> a + b }

fun main() {
    val wynik = dodaj(3, 4)
    println(wynik)  // Wypisze: 7
}
```

## Lambda bez parametrów
```kotlin
val powitaj = { println("Cześć Kotlin!") }

fun main() {
    powitaj()  // Wypisze: Cześć Kotlin!
}
```

## Lambda z wyrażeniem warunkowym
```kotlin
val maks = { a: Int, b: Int -> if (a > b) a else b }

fun main() {
    println(maks(10, 20))  // Wypisze: 20
}
```

# Skrócone lambdy
Automatyczne rozpoznawanie typów
```kotlin
val dodaj = { a: Int, b: Int -> a + b }  // Z pełnymi typami
val dodajProsta: (Int, Int)->Int = { a, b -> a + b }  // Kotlin wywnioskuje, że to Int


fun main() {
    println(dodaj(11,22))  // Wypisze: 33
}
```

# Użycie `it`
- Jeśli lambda przyjmuje tylko jeden argument, nie trzeba go jawnie nazywać.
- Kotlin automatycznie przypisuje ten argument do zmiennej it.

```kotlin
val podwojZItInput = { it: Int -> it * 5 }
val podwojBezItInput: (Int)->Int = { it * 5 }

fun main() {
    println(podwojZItInput(5))  // Wypisze: 25
    println(podwojBezItInput(5)) // Wypisze 25
```

# Użycie lambd z funkcjami wyższego rzędu
- Funkcje wyższego rzędu to takie, które
    - mogą przyjmować inne funkcje jako argumenty lub
    - zwracać funkcje.

W Kotlinie lambdy są często używane z funkcjami wyższego rzędu, aby pisać bardziej zwięzły i elastyczny kod.

## Przykład
Funkcja wyższego rzędu, która przyjmuje dwie liczby i operację
```kotlin
fun operujNaLiczbach(a: Int, b: Int, operacja: (Int, Int) -> Int): Int {
    return operacja(a, b)
}

fun main() {
    // Przykład użycia lambdy do dodawania
    val suma = operujNaLiczbach(3, 4) { x, y -> x + y } // jeżeli lambda jest ostatnim parametrem,
                                                        // można ją napisać poza nawiasem
                                                        // a jeżeli jest jedynym parametrem, można pominąć nawiasy zwykłe ()
    println("Suma: $suma")  // Wypisze: Suma: 7

    // Przykład użycia lambdy do mnożenia
    val iloczyn = operujNaLiczbach(3, 4) { x, y -> x * y }
    println("Iloczyn: $iloczyn")  // Wypisze: Iloczyn: 12
```
