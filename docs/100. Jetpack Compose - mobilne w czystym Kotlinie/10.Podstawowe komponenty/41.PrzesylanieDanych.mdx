# Przesyłanie dany między komponentami *Compose*

:::important ZŁOTA ZASADA
In Jetpack Compose:
- data flows down the tree (from parent to children), and
- events flow up (from children to parents).

This is called unidirectional data flow (UDF).

- Dane płyną w dół (z rodzica do dziecka),
- a zdarzenia płyną w górę (z dziecka do rodzica).

:::
