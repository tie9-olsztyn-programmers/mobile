  • Wprowadzenie do klas w Kotlinie

Klasy i obiekty w Kotlinie:

  • Klasa to definicja, schemat, na podstawie którego tworzysz obiekty.
  • Obiekt to konkretna instancja klasy.

Metaforyczne wyjaśnienie​

  • klasa jest jak przepis na ciasto:
    • przepis dokładnie opisuje,
      • jakie składniki są potrzebne i
      • jak krok po kroku upiec ciasto —
    • ale sam przepis nie jest jeszcze ciastem, po prostu mówi, jak je zrobić.
  • obiekt:
    • to gotowe ciasto, które powstało na podstawie przepisu,
    • możesz upiec wiele ciast według tego samego przepisu,
    • ale każde ciasto to już konkretny obiekt — można je zjeść, pokroić, udekorować inaczej itd.
// To jest klasa, czyli przepis na ciasto
class Cake(
    val flavor: String,   // smak ciasta
    val layers: Int       // liczba warstw
) {
    // To jest metoda obiektu.
    // Każdy obiekt tej klasy będzie potrafił wydrukować w konsoli
    // podstawowe informacje o sobie samym
    fun describe() {
        println("Ciasto o smaku $flavor, liczba warstw: $layers")
    }
}

// Tworzymy obiekty, czyli gotowe ciasta według przepisu (klasy)
fun main() {
    val chocolateCake = Cake("czekoladowy", 3)
    val vanillaCake = Cake("waniliowy", 2)

    chocolateCake.describe() // wypisze: Ciasto o smaku czekoladowy, liczba warstw: 3
    vanillaCake.describe()   // wypisze: Ciasto o smaku waniliowy, liczba warstw: 2
}

Definicja​

W Kotlinie (i programowaniu obiektowym):

  • Klasa to opis tego, jak powinien wyglądać i co powinien potrafić dany "rodzaj rzeczy" (np. samochód, użytkownik, ciasto).
  • Obiekt to już gotowa, istniejąca rzecz (instancja) utworzona na podstawie klasy (czyli przepisu).

Definiowanie klasy​

W Kotlinie definicja klasy jest bardzo prosta. Używamy słowa kluczowego class, aby zdefiniować klasę:

class Osoba {
    // Zmienne instancyjne
    var imie: String = ""
    var wiek: Int = 0

    // Metoda w klasie
    fun powitaj() {
        println("Cześć, mam na imię $imie i mam $wiek lat.")
    }
}

Tworzenie obiektu​

class Osoba {
    // Zmienne instancyjne
    var imie: String = ""
    var wiek: Int = 0

    // Metoda w klasie
    fun powitaj() {
        println("Cześć, mam na imię $imie i mam $wiek lat.")
    }
}


fun main() {
    // Tworzenie obiektu klasy Osoba
    val osoba = Osoba()
    osoba.imie = "Jan"
    osoba.wiek = 30

    // Wywołanie metody obiektu
    osoba.powitaj()  // Wypisze: Cześć, mam na imię Jan i mam 30 lat.
}
