# Typy pętli
W Kotlinie dostępne są trzy główne rodzaje pętli:
- `for`
- `while`
- `do-while`

umożliwiają wykonywanie tego samego fragmentu kodu wielokrotnie, aż do spełnienia określonego warunku.

# Pętla for

```kotlin
for (element in collection) {
    // kod wykonywany dla każdego elementu
}
```
- *element* to zmienna, która w każdej iteracji reprezentuje jeden z elementów kolekcji.
- *collection* to iterowalna struktura danych, np. lista, zakres, tablica itp.

```kotlin
fun main() {

    for (i in 1..5) {

    println(i)  // Wypisuje liczby od 1 do 5

    }

}
```

```kotlin
fun main() {

    val names = arrayOf("Anna", "Kamil", "Jan")

    for (name in names) {

        println(name)
    }
}
```

```kotlin
fun main() {

    for (i in 1..10 step 2) {

        println(i)  // Wypisuje: 1, 3, 5, 7, 9

    }
}
```

```kotlin
fun main() {

    for (i in 5 downTo 1) {

        println(i)  // Wypisuje: 5, 4, 3, 2, 1

    }
}
```

# Pętla while

```kotlin
while (condition) {
    // kod wykonywany, dopóki condition jest true
}
```


```kotlin
fun main() {

    var i = 5

    while (i > 0) {

        println(i)

        i--
    }
}
```

# Pętla do-while

```kotlin
do {
    // kod wykonywany przynajmniej raz
} while (condition)
```

```kotlin
fun main() {

    var i = 5

    do {

        println(i)

        i--

    } while (i > 0)
}
```

`break` - przerywa działanie pętli w dowolnym momencie

`continue` - powoduje pominięcie bieżącej iteracji pętli i przejście do następnej
